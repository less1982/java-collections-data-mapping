# java-collections-data-mapping
Demonstrates efficient data management using Java Collections (HashMap) to store, retrieve, and manipulate key-value pairs with O(1) search complexity.

The Goal: To demonstrate the implementation of the HashMap collection to store and manipulate key value pairs (Country Codes and Names) with high search efficiency.

The Challenge: Managing dynamic data sets where entries need to be updated, removed, and iterated through without losing data integrity or type safety.

The Solution: I utilized Java Generics to ensure type safety and implemented Map.Entry iteration to efficiently display and update the state of the data structure after modifications.
